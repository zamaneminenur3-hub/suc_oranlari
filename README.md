# 📊 Suç Oranları Projesi

Türkiye mahkumlar veri seti kullanılarak oluşturulan profesyonel görselleştirme ve analiz projesi.

## 🎯 Proje Hakkında

Bu proje, Türkiye'deki mahkumlara ilişkin kapsamlı veri analizi sunan profesyonel bir raporlama sistemidir. Python ve veri analiz araçları kullanılarak oluşturulan etkileşimli görselleri ile suç türleri, demografik dağılımlar ve zamansal trendleri detaylı bir şekilde ortaya koymaktadır.

## 📈 İçeriğindeki Grafikler

1. **Yıllara Göre Toplam Mahkum Sayısı** - Çizgi grafik ile yıllık eğilimleri gösteren analiz
2. **En Sık İşlenen 10 Suç Türü** - En yaygın suç kategorilerinin dağılımı
3. **Cinsiyete Göre Mahkum Dağılımı** - Erkek ve kadın mahkum oranları
4. **Yaş Gruplarına Göre Mahkum Dağılımı** - Yaş stratifikasyonu analizi
5. **Yıllar İçinde En Sık 5 Suçun Değişimi** - Suç trendlerinin zamansal değişimi

## 🛠️ Teknoloji Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Data Processing**: Python, Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn
- **Icons**: Font Awesome 6.4
- **Hosting**: GitHub Pages

## 🚀 Nasıl Kullanılır

### Yerel Olarak Açmak

#### Yöntem 1: Doğrudan Açma
```bash
open index.html
```

#### Yöntem 2: Python Sunucusu ile
```bash
cd suc_oranlari
python3 -m http.server 8000
```
Ardından tarayıcıda `http://localhost:8000` ziyaret edin.

#### Yöntem 3: VS Code Live Server
1. VS Code'da `index.html` açın
2. Sağ tıklayıp "Open with Live Server" seçin
